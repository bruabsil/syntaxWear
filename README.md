# SyntaxWear

Projeto de landing page HTML + CSS focado em moda techwear e estrutura semântica.

## Estrutura

- `index.html`
- `css/base.css`
- `css/layout.css`
- `css/components/`
- `images/`
- `icons/`
- `fonts/`
- `favicons/`
- `docs/`

## Como usar

1. Abra `index.html` no navegador.
2. Edite os estilos em `css/base.css` e `css/layout.css`.
3. Adicione componentes CSS dentro de `css/components/`.
